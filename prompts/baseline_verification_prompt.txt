5. Makefile  (must support the metrics the VerifEval paper measures)
================================================================================

The Makefile must define BOTH a Questa flow (preferred — same as paper) and a
Verilator flow (fallback for environments without a commercial license). Both
flows must accept the same `make compile`, `make sim`, `make coverage`,
`make report` targets. Use a `SIM ?= questa` switch.

Required targets:

  make sim         # run all 5 tests, dump waves to waves/${DUT}.vcd
  make coverage    # run with coverage instrumented:
                   #   Questa:    vsim -coverage -do "coverage save ..."
                   #              vcover report -details
                   #   Verilator: --coverage --trace-fst then verilator_coverage
  make lint        # verible-verilog-lint over tb/, agents/, assertions/
  make report      # write reports/coverage_summary.json with EXACTLY these keys:
                   #   {
                   #     "dut": "${DUT}",
                   #     "simulator": "questa|verilator",
                   #     "metrics": {
                   #        "assertions_pct": <float 0-100>,
                   #        "branches_pct":   <float 0-100>,
                   #        "conditions_pct": <float 0-100>,
                   #        "directives_pct": <float 0-100>,
                   #        "statements_pct": <float 0-100>,
                   #        "toggles_pct":    <float 0-100>,
                   #        "overall_pct":    <float 0-100>
                   #     },
                   #     "tests_run":    <int>,
                   #     "tests_passed": <int>,
                   #     "tests_failed": <int>,
                   #     "log":          "<relative path to sim log>"
                   #   }
  make clean

The 7 keys under `metrics` MUST always be present (use null only if a metric
is not produced by the chosen simulator). They mirror the columns of Table 1
in the VerifEval paper so the result can be dropped straight into the
benchmark report.
